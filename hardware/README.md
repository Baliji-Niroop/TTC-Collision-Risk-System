# Hardware Documentation

This folder contains physical assembly and wiring guidance for TTC hardware.

## Files

- `assembly_checklist.md`: Procurement, assembly, upload, and bring-up checklist.
- `wiring_guide.md`: Complete pin map and wiring reference.

## Source of truth

- Firmware pin assignment: `firmware/pinmap.h`
- Firmware protocol contract: `firmware/config/serial_protocol.h`

## Read in this order

1. wiring_guide.md
2. assembly_checklist.md
