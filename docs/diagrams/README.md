# Diagram Sources

These Mermaid sources are the canonical report-ready diagram inputs for the TTC project.

- `architecture.mmd` for the software architecture overview.
- `telemetry_protocol.mmd` for the 7-field packet contract.
- `ttc_flowchart.mmd` for the TTC processing flow.
- `state_machine.mmd` for the risk state transitions.

Render them with any Mermaid-compatible renderer and export the resulting PNG/SVG into the report bundle.
